# Checklist: Iniciacao de Projeto (PMBOK)

## Metadata
- **id**: project-initiation-checklist
- **version**: 1.0.0
- **pmbok_phase**: Initiating

## Pre-Requisitos
- [ ] Demanda/necessidade de negocio identificada
- [ ] Sponsor do projeto definido
- [ ] Viabilidade inicial avaliada

## Project Charter (Termo de Abertura)
- [ ] Proposito/justificativa do projeto documentado
- [ ] Objetivos SMART definidos
- [ ] Requisitos de alto nivel listados
- [ ] Descricao de alto nivel do projeto
- [ ] Premissas documentadas
- [ ] Restricoes documentadas
- [ ] Riscos de alto nivel identificados
- [ ] Marcos (milestones) principais definidos
- [ ] Orcamento resumido estimado
- [ ] Criterios de sucesso definidos
- [ ] Sponsor assinou/aprovou o charter

## Stakeholders (Partes Interessadas)
- [ ] Stakeholders identificados
- [ ] Registro de stakeholders criado
- [ ] Grid Poder/Interesse preenchido
- [ ] Estrategia de engajamento inicial definida

## Governanca
- [ ] Gerente de projeto designado
- [ ] Autoridade do GP definida
- [ ] Estrutura de governanca estabelecida
- [ ] Comite de projeto definido (se aplicavel)

## Transicao para Planejamento
- [ ] Charter aprovado pelo sponsor
- [ ] Kickoff meeting agendado
- [ ] Equipe core notificada
- [ ] Ambiente de trabalho preparado
- [ ] Ferramentas de GP selecionadas

## Notas
> Use `@pm-chief *charter` para gerar o Project Charter automaticamente.
> Use `@pm-chief *stakeholders` para criar o Registro de Stakeholders.
