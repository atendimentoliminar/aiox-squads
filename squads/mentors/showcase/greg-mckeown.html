Framework Central

O Funil Essencialista

Essentialism não é sobre fazer mais coisas — é sobre fazer as coisas certas. Três fases: explorar, eliminar, executar.

🔍
Fase 1 — DISCERNIR
Explorar e Avaliar

Crie espaço para pensar e explorar as opções antes de se comprometer. Distinga as poucas coisas vitais das muitas triviais. Explore, questione, ouça. O Essencialista nunca abre mão do direito de explorar.

"De todas as opções, qual é o mais alto ponto de contribuição?"
✂️
Fase 2 — ELIMINAR
Cortar Sem Piedade

Eliminar é a parte mais difícil e mais importante. Cada sim automático é um não implícito para o que realmente importa. O não precisa ser claro — não desculpas, não atrasos.

A Regra dos 90%: se não é um SIM claro (90+), é um NÃO.
⚡
Fase 3 — EXECUTAR
Tornar a Execução sem Esforço

Remova os obstáculos que tornam o essencial difícil. Crie sistemas que fazem o progresso o caminho de menor resistência. Proteja o ativo — você é o mais importante.

"O que isso pareceria se fosse fácil?"
A Decisão Binária

A Regra dos 90%

Se não for um SIM claro de 90+, é um NÃO. A zona cinzenta entre sim e não é onde o medíocre vive.

NÃO-ESSENCIALISTA
"Parece razoável…"
  • Diz sim para evitar desconforto
  • Avalia de 0–100 e aceita qualquer coisa acima de 60
  • Mantém opções abertas — dificuldade em comprometer
  • Agenda cheia de "boas" oportunidades
  • Resultado: progresso difuso, impacto medíocre
→
ESSENCIALISTA
"É um SIM claro ou é um NÃO."
  • Define critérios mínimos para qualquer sim
  • Avalia de 0–100: apenas 90+ passa automaticamente
  • O NÃO é a posição padrão — o sim requer justificativa
  • Agenda protegida para o essencial
  • Resultado: impacto profundo nas poucas coisas que importam
A Decisão das 1000 Decisões

Essential Intent

Uma intenção essencial é tão clara que elimina automaticamente mil outras decisões.

Definição
Uma decisão que elimina 1.000 decisões

Essential Intent é uma frase única: inspiradora o suficiente para motivar, concreta o suficiente para ser testada. Quando está certo, você sabe imediatamente se qualquer nova oportunidade é essencial ou não.

Sem Essential Intent
"Queremos ser os melhores em tudo que fazemos e servir nossos clientes com excelência em todas as dimensões possíveis."
Com Essential Intent
"Construir ON AI Solutions como a primeira plataforma de agentes de IA para empreendedores brasileiros, até 2026, gerando receita previsível sem equipe operacional."
Princípios Fundamentais

A Filosofia do Essencialismo

01 / MENOS, MELHOR
A Busca Disciplinada pelo Menos

Essentialism não é fazer menos por preguiça — é fazer menos, melhor. "Onde está meu mais alto ponto de contribuição?" Responda. Faça apenas isso.

02 / ESCOLHA
Você Sempre Tem Escolha

"Tenho que fazer isso" é uma mentira. Sempre há escolha. Delegar a escolha para convenções ou para o urgente ainda é uma escolha — e uma escolha ruim.

03 / TRADEOFF
Tradeoffs São Reais

Trade-offs são reais. A questão não é como evitá-los — é quais aceitar conscientemente. "Qual problema quero ter?"

04 / EFFORTLESS
O Caminho Mais Fácil

'O que isso pareceria se fosse fácil?' desfaz o reflexo do esforço bruto. Lento é suave. Suave é rápido.

05 / PROTEGER
Proteja o Ativo

Você é o ativo mais importante. Energia, atenção e saúde são pré-requisitos do trabalho essencial — não luxos. Dormir é produtivo.

06 / PROGRESSO
Minimum Viable Progress

"Qual é o menor progresso possível que me move para frente?" Progresso mínimo viável gera momentum sem paralisação.

"Se não for um SIM claro, é um NÃO. Se você tivesse que decidir agora, sem 'talvez' ou condições, qual seria a resposta? Se houver qualquer hesitação — é um não."
