Princípios

As 7 Regras Inegociáveis

Cada princípio foi testado na prática — inclusive errando primeiro (30h de áudio sem curadoria que virou clone genérico) e depois acertando (clone Hormozi, R$2.1M documentados).

01 / CURADORIA É TUDO
Qualidade da Entrada Define Qualidade da Saída

Uma fonte ouro vale mais que 30 horas de conteúdo bronze. Volume sem curadoria não é riqueza de material — é ruído com escala. A curadoria não é etapa opcional: é onde mora a fidelidade do clone.

02 / CLONE MINDS, NÃO BOTS
Replica o Raciocínio, Não o Texto

Pessoas reais têm skin in the game. Seus frameworks foram testados na realidade, não em sala de aula. Clone de mente real > bot genérico com prompt de personagem. A fidelidade importa porque o que você replica é o raciocínio.

03 / A TRINDADE É SAGRADA
Playbook + Framework + Swipe File

Clone precisa de Playbook (O QUE fazer) + Framework (QUANDO/COMO decidir) + Swipe File (exemplos reais). Falta qualquer um dos três — clone fraco. Playbook sem Framework é só receita. Clone precisa decidir, não só executar.

04 / PARETO AO CUBO
80/20 Aplicado 3 Vezes

0,8% das atividades produzem ~51% do resultado. A pergunta certa não é 'o que devo fazer?' — é 'o que devo parar de fazer?' A Zona de Merda (80%) precisa ser eliminada agressivamente, sem pena, para a Zona de Genialidade aparecer.

05 / RASTREABILIDADE
Sem [SOURCE:] Não Existe

Cada conceito extraído precisa de [SOURCE: página/minuto]. Se não tem fonte, é inferência — marque [INFERRED] ou não use. Clone que inventa parece bom no início e falha no momento que importa. Se não sobrevive ao reset, não tá documentado.

06 / 40/20/40
Distribuição Correta do Esforço

40% do esforço vai em curadoria. 20% em prompt/estrutura. 40% em refinamento. Inverter essa ordem é o erro mais comum — e o mais caro. Quem pula a curadoria para chegar logo no prompt está construindo em areia.

07 / REUSE ANTES DE CRIAR
Quem Já Faz Isso Bem?

Antes de criar qualquer framework, task ou sistema: 'Quem já faz isso bem?' Adaptar o que foi validado bate inventar do zero em 90% dos casos. A pesquisa antes da criação não é burocracia — é respeito pelo trabalho existente.

Se não sobrevive ao reset, não tá documentado — tá só na sua cabeça.
