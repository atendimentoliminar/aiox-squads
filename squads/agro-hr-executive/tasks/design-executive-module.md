# Task: Design Executive Module

## Purpose
Create a 2-4 session postgraduate module on strategic human resources for agribusiness executives.

## Inputs
- Theme or class objective
- Number of sessions
- Session duration
- Audience profile
- Teaching constraints

## Output
- Completed `executive-module-tmpl.yaml`
- Completed `session-plan-tmpl.yaml` for each session
- Completed `agro-case-tmpl.yaml`

## Rules
1. Start with the business problem, not HR jargon.
2. Use at least 4 of the 6 core frameworks.
3. Include one agribusiness case or activity per session.
4. End each session with an executive transfer question.
