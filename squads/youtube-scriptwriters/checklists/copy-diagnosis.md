# Copy Diagnosis Checklist

Diagnostic checklist for auditing scripts and copy before rewriting.
Used by `neville-medhora` (*diagnose, *checklist).

## How to Use
Run this checklist on any script/copy BEFORE attempting to rewrite.
Diagnosis BEFORE prescription — always.

---

## 1. Bob Test (Audience Targeting)

| # | Check | Status | Notes |
|---|-------|--------|-------|
| 1.1 | Written to ONE specific person (Bob), not "pessoal" or "vocês" | | |
| 1.2 | Bob's specific pain/desire is named (not generic) | | |
| 1.3 | Uses Bob's language (words he would actually say) | | |
| 1.4 | Every sentence would make Bob lean in | | |

**If FAIL:** Define Bob first (name, age, job, fear, desire, objection, exact words).

---

## 2. AIDA Flow Analysis

| Phase | Present? | Quality | Diagnostic |
|-------|----------|---------|------------|
| **Attention** (first 3-5s) | | | If retention drops in first 5s → Attention is broken |
| **Interest** (5-30s) | | | If retention drops at 30s → Interest is broken |
| **Desire** (body) | | | If engagement is flat through middle → Desire is broken |
| **Action** (CTA) | | | If CTA conversion is low → Action broken or Desire wasn't built |

**Identify which AIDA phase is broken** — this determines the fix.

**Diagnostic test per sentence:**
1. Would I say this exact sentence to a friend? (NO = Boring)
2. Does it sound like a corporate email? (YES = Boring)
3. Is there a simpler way to say this? (YES = simplify)
4. Am I using jargon Bob wouldn't use? (YES = Boring)

## 4. 70/30 Ratio Check

Count sentences in the script:

| Type | Count | Percentage |
|------|-------|------------|
| Education / Value | | |
| Pitch / Promotion | | |

**Healthy:** 70% education, 30% pitch (or better)
**Salesy:** Below 70/30 — too much pitching
**Leaving money on table:** Above 90/10 — not enough asking

| 6 | Benefits stated, not just features? | |

| 8 | Would YOU read this if someone else wrote it? | |

**Score:** /10
- 8-10: Ship it

- Below 5: Rewrite from diagnosis
